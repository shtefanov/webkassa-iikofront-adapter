# Code of Conduct

This project is maintained for professional software development and customer
operations.

Participants are expected to:

- communicate respectfully and directly;
- keep technical discussions focused on evidence and reproducible behavior;
- protect customer data and secrets;
- avoid publishing private operational details outside approved channels;
- document risks, validation, and rollback paths clearly.

Unacceptable behavior includes harassment, leaking secrets, sharing customer
data without approval, or deliberately bypassing security controls.

For private repository work, report conduct or security concerns to the
repository owner.
