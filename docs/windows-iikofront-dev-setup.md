## Confirmed API References

Official iikoFront docs confirm external fiscal register path:

- implement `ICashRegisterFactory`;
- register it via `PluginContext.Operations.RegisterCashRegisterFactory(...)`;
- factory creates `ICashRegister`;
- iikoFront calls `ICashRegister.DoCheque(ChequeTask, IViewManager)` when an
  order is closed or storned on the plug-in cash register;
- `DoCheque` returns `CashRegisterResult`;
- `ChequeTask.IsRefund` distinguishes return flow.

`IChequeTaskProcessor` is useful for pre/post hooks around existing fiscal
registers, but it is not the primary replacement for Webkassa fiscalization.
